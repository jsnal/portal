# portal

Source code for [jasonlong.xyz](http://jasonlong.xyz).

## Development

### Project setup
```
$ npm install
```

### Development server
```
$ npm run dev
```

### Lints and fixes files
```
$ npm run lint
```

## Deployment

### Build the Dockerfile
```
# docker build . -t portal
```

### Run the container
```
# docker run -d -p 80:80 portal
```
